### Added. Upgrades the platform to Spring Boot 4.0.7
<!-- CXBOX-1319 -->

This release upgrades the platform to:

* Spring Boot 4.0.7
* Hibernate 7.2.19


**Since Spring Boot  3.5 will receive open-source updates only until [June 30, 2026](https://endoflife.date/spring-boot), 
we are upgrading the project to Spring Boot  4.0.7 and Hibernate ORM 7.2.19 to ensure continued support and access to the latest features and improvements**
 
**We strongly recommend performing the upgrade in the following sequence** see more [Spring Boot 4.0 Migration Guide](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-4.0-Migration-Guide)

1) Step 1. Upgrade to [CXBOX 4.0.0-M25](https://github.com/CX-Box/cxbox/tree/cxbox-4.0.0-M25) [CXBOX Migration Guide](/new/migrationguide3515/)
 
2) Step 2. Upgrade to CXBOX 5.0.0
